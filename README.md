    <h1>🌱 AgroSense</h1>
    <p>
        AgroSense is a comprehensive solution designed to empower farmers by providing direct market access for their crops. This project, part of our college's CP-1 initiative, aims to eliminate intermediaries, improve profitability, and provide tools for enhanced agricultural practices.
    </p>

    <div class="section">
        <h2>🚀 Features</h2>
        <ul>
            <li>🌾 Direct Market Access: Farmers can sell crops directly to buyers.</li>
            <li>💡 AI Chatbot Support: Assistance for agricultural queries.</li>
            <li>📚 Learning Resources: Educational materials for farmers.</li>
            <li>🏛️ Government Schemes: Access to beneficial schemes and policies.</li>
            <li>🧑‍🏫 Expert Guidance: Help from agricultural specialists.</li>
            <li>🌱 AI Crop Problem Detection: Diagnose issues with crops using AI.</li>
        </ul>
    </div>
